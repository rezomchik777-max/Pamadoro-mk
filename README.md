# Pomodoro Kombat

Android-приложение Pomodoro-таймера в ретро-боевой эстетике.

Фичи:
- welcome/onboarding экран
- режимы 25 / 15 / 50 минут и кастом 5–90 минут
- русский/английский интерфейс
- ретро-аркадные звуки без copyrighted Mortal Kombat assets
- вибрация
- уведомление после завершения раунда
- XP, ранг, победы, серии дней, общее время фокуса
- темы Scorpion / Sub-Zero
- AMOLED dark mode
- экран Missions/Settings
- экспорт статистики через системное меню Android

Сборка APK:
1. Открой проект в Android Studio.
2. Дождись Gradle Sync.
3. Build → Build APK(s).
4. APK будет в `app/build/outputs/apk/debug/app-debug.apk`.
